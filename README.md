# httpsDXF — personal site

Next.js (App Router) + Tailwind. Primary domain: [httpsdxf.me](https://httpsdxf.me).

## Commands

```bash
npm install
npm run dev
npm run build   # static export → out/ (GitHub Pages)
```

## Structure

- `app/` — pages, layouts, components
- `app/config/` — site, navigation, footer links
- `public/` — static assets, `CNAME` for custom domain
